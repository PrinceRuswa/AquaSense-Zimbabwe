Step-by-Step Assembly Guide

1
Gather & Verify Components
Before soldering, test each component on a breadboard. Use a multimeter to check voltages.
✅ ESP32 boots and connects to Arduino IDE
✅ Soil sensor reads 0–3V analog output
✅ DHT22 responds to digital reads
✅ LoRa module detected on SPI bus
✅ Solar panel outputs 5–7V in sunlight
✅ TP4056 charges 18650 correctly
2
Breadboard Prototype & Test
Wire all components on a breadboard first. Flash test firmware and verify all sensor readings.
⚡ Key Tests:
• Serial monitor shows soil % and temp/humidity values
• LoRa transmits packets (check with serial plotter)
• Deep sleep current drops to <1mA (measure with meter)
• Power system charges battery from solar panel
3
Solder on Perfboard / PCB
Transfer from breadboard to perfboard using solid-core wire. Keep power rails clean.
📌 Use 22AWG silicone wire for flexibility
📌 Solder TP4056 and MT3608 separately on sub-board
📌 Use pin headers for ESP32 (removable for flashing)
📌 Add 100nF caps near each VCC pin
📌 Label all wire ends with heat-shrink marker
4
Prepare IP65 Enclosure
Drill holes for cable glands, LED indicator, and antenna SMA connector.
🔧 Drill 16mm hole for soil sensor cable gland (bottom)
🔧 Drill 16mm hole for solar panel cable entry (top)
🔧 Drill 5mm hole for SMA antenna connector (side)
🔧 Drill 4mm hole for 3mm LED indicator (lid)
🔧 Apply silicone sealant around all penetrations
🔧 Mount DIN rail or standoffs inside for PCB
5
Install PCB in Enclosure
Mount the PCB using M3 nylon standoffs. Secure the 18650 battery holder.
📦 PCB mounts on 4× M3×10mm nylon standoffs
📦 Battery holder fixed with M2 screws to enclosure base
📦 Connect antenna SMA through drilled hole
📦 Feed soil sensor cable through cable gland
📦 Tighten cable glands to maintain IP65 rating
6
Prepare & Install Field Post
Drive treated wood post 40cm into soil. Mount enclosure and solar panel on post.
🪵 Post should stand 80cm above soil level
🪵 Mount enclosure at 60–70cm height (avoid splash)
🪵 Solar panel at top, tilted 15–20° facing North (Zimbabwe)
🪵 Use galvanised L-bracket + M6 stainless bolts
🪵 Apply wood preservative / used engine oil to post base
7
Insert Soil Probe & Calibrate
Insert soil sensor probe 5–8cm deep near plant root zone. Calibrate for your soil type.
🌱 Insert probe at 45° angle, 5–8cm depth
🌱 Take reading in bone-dry soil (note raw ADC value ≈ 2950)
🌱 Submerge probe in water (note raw ADC value ≈ 1200)
🌱 Map these to 0–100% in firmware map() function
🌱 Set crop thresholds: Maize=30-70%, Tobacco=25-65%
8
Final Power-On & Cloud Verify
Power on the node and verify data appears in the AquaSense dashboard.
✅ Green LED blinks on each LoRa transmission
✅ Dashboard shows new sensor node appearing
✅ Soil moisture, temperature, humidity readings live
✅ Battery level reporting correctly
✅ Node enters deep sleep between transmissions
Tools Required
🔧 Soldering iron (30–40W) ⚡ Multimeter 💻 Laptop with Arduino IDE 🪛 Screwdrivers (Phillips + flat) 🔩 Drill + 4/5/16mm bits ✂️ Wire stripper + cutters 🔥 Heat gun / lighter (heat shrink) 🔬 Breadboard + jumper wires

RAK7268 LoRaWAN Gateway Setup

RAK7268 WisGate Edge Lite 2
8-channel indoor LoRaWAN gateway with Ethernet + optional 4G LTE
Channels: 8 (configurable)
Frequency: 915MHz (AU915)
TX Power: 27dBm
Range: ~10km open field
Backhaul: Ethernet + 4G LTE
Protocol: LoRaWAN 1.0.3
Power: 12V DC adapter
Housing: Plastic, indoor

Gateway Installation Steps:

1. Mount gateway in farmhouse/barn, connect Ethernet to router OR insert Econet/NetOne SIM for 4G
2. Access gateway web UI at 192.168.1.1, configure LoRa server to point to The Things Network (TTN) — free cloud LoRaWAN server
3. Register gateway on TTN console (console.thethingsnetwork.org), add each sensor node as TTN device with OTAA keys
4. Set up TTN webhook to POST sensor data to AquaSense Zimbabwe dashboard API endpoint
5. Install outdoor 5dBi antenna on roof, connect via SMA cable for extended 10km coverage across farm

Firmware Sample Code (Arduino/ESP32)

// AquaSense Zimbabwe — Sensor Node Firmware
// ESP32 + LoRa + DHT22 + Soil Moisture

#include <LoRa.h>
#include <DHT.h>
#include <esp_sleep.h>

// Pin Definitions
#define SOIL_PIN 34 // ADC GPIO34
#define DHT_PIN 4 // GPIO4
#define LORA_SCK 18
#define LORA_MISO 19
#define LORA_MOSI 23
#define LORA_NSS 15
#define LORA_RST 2
#define LORA_DIO0 26
#define SLEEP_US 15 * 60 * 1000000

const char* FIELD_ID = "F1";

DHT dht(DHT_PIN, DHT22);

void setup() {
Serial.begin(115200);
dht.begin();
// Init LoRa
LoRa.setPins(LORA_NSS, LORA_RST, LORA_DIO0);
if (!LoRa.begin(915E6)) {
Serial.println("LoRa FAIL");
return;
}
sendData(); // Send readings
LoRa.end();
// Deep sleep for 15 min
esp_deep_sleep(SLEEP_US);
}

void sendData() {
int rawSoil = analogRead(SOIL_PIN);
int moisture = map(rawSoil,2950,1200,0,100);
moisture = constrain(moisture, 0, 100);
float temp = dht.readTemperature();
float hum = dht.readHumidity();

// Build JSON payload
String payload = "{";
payload += "\"id\":\"" + String(FIELD_ID) + "\",";
payload += "\"m\":" + String(moisture) + ",";
payload += "\"t\":" + String(temp) + ",";
payload += "\"h\":" + String(hum);
payload += "}";

LoRa.beginPacket();
LoRa.print(payload);
LoRa.endPacket();
Serial.println(payload);
}

void loop() {} // never runs (deep sleep)
Libraries needed (Arduino IDE): arduino-LoRa by Sandeep Mistry, DHT sensor library by Adafruit. Install via Sketch → Include Library → Manage Libraries
